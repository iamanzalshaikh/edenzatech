# EDENZATECH

Software development company website — **INNOVATE • BUILD • GROW**

Built with Next.js 16, React 19, Tailwind CSS 4, Framer Motion, GSAP, and Lenis smooth scroll.

## Brand

- **Accent:** Cyan `#00E5FF` → Blue `#0052FF` gradient
- **Background:** Dark black `#050508`
- **Logo:** `public/logo-edenzatech.png`

## Services (development only)

- Web Development
- Mobile App Development
- Custom Software & SaaS
- E-Commerce Development
- API & Backend Development
- UI/UX & Frontend

Marketing services (SEO, social media, ads, branding) are commented out in `src/lib/site-config.ts`.

## Commands

```bash
npm install
npm run dev    # http://localhost:3000
npm run build
```

## Structure

```
src/
├── app/              # Pages & layout
├── components/
│   ├── home/         # Homepage sections
│   ├── layout/       # Header, footer
│   └── ui/           # Buttons, counters, FAQ
└── lib/              # Config, data, assets
```
